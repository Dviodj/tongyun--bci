## 🔧 技术方案

### 方法1: EEGNet预训练 (77.8%)
1. **预训练**: 在全部25位被试数据上预训练EEGNet深度学习模型
2. **特征提取**: 通过卷积层自动学习运动想象特征
3. **评估**: 留一被试(LOSO)评估

### 方法2: 传统特征+LDA (71.5%)
1. **特征提取**: mu频段(8-13Hz)和beta频段(13-30Hz)功率
2. **空间特征**: C3/C4不对称性分析
3. **分类器**: LDA (线性判别分析)

### 数据预处理
- 0.5-45Hz 带通滤波
- 平均参考
- Baseline校正: [-0.5s, 0s]
- Epoch窗口: [-0.5s, +3.5s]
- 通道: C3, Cz, C4

## 📝 摩斯密码映射

| 信号类型 | 脑电信号 | 摩斯符号 |
|---------|---------|---------|
| 左手运动想象 | 左脑(C3)激活增强 | `.` (点) |
| 右手运动想象 | 右脑(C4)激活增强 | `-` (划) |

### 示例：HELLO
```
原文: H  E  L  L  O
摩斯: .... . .-.. .-.. ---
信号: 左左左左  左  左右左左  左右左左  左左左左
```

## 📖 方法说明

### 为什么Cross-subject > Within-subject？
BCICIV 2b每位被试仅有120-160个epochs，样本量极小。Within-subject CV每折仅96-128训练样本，严重欠拟合。Cross-subject LOSO用其他被试数据训练，数据量充足，因此泛化性能更真实。

### 关于99.1%结果的说明
早期测试在以下三处发生数据泄露：
1. CSP在全部数据上拟合，验证集参与了空间滤波器训练
2. StandardScaler在全部数据上拟合
3. 超参数在泄漏特征上搜索

现已修复。所有特征提取和模型训练均在严格划分的训练集上进行。
