# 🧠 脑电信号 → 摩斯密码识别系统

通过识别左右手脑电信号，转换为摩斯密码，最终输出文字。

## 🔧 技术方案

### 数据加载
- 使用 **MNE** 库直接加载 GDF 格式数据
- 预处理：0.5-45Hz 带通滤波 + 平均参考

### 特征提取
1. **mu频段(8-13Hz)功率** - 运动想象核心特征
2. **beta频段(13-30Hz)功率** - 辅助判别
3. **C3/C4 不对称性** - 左右手运动想象的关键区分特征
4. **时域统计量** - 均值、标准差

### 分类器
- **LDA (线性判别分析)** - EEG分类最常用且稳健的分类器
- **SVM-rbf** - 备选

## 📝 摩斯密码映射

| 信号类型 | 脑电信号 | 摩斯符号 |
|---------|---------|---------|
| 左手运动想象 | 左脑(C3)激活增强 | `.` (点) |
| 右手运动想象 | 右脑(C4)激活增强 | `-` (划) |

### 示例：HELLO
```
原文: H  E  L  L  O
摩斯: .... . .-.. .-.. ---
信号: 左左左左  左  左右左左  左右左左  左左左左
```

## 📖 方法说明

### 为什么Cross-subject > Within-subject？
BCICIV 2b每位被试仅有120-160个epochs，样本量极小。Within-subject CV每折仅96-128训练样本，严重欠拟合。Cross-subject LOSO用25个被试(≈3400样本)训练一位被试，数据量充足，因此泛化性能更真实。

### 关于99.1%结果的说明
早期测试在以下三处发生数据泄露：
1. CSP在全部数据上拟合，验证集参与了空间滤波器训练
2. StandardScaler在全部数据上拟合
3. 超参数在泄漏特征上搜索

现已修复。所有特征提取和模型训练均在严格划分的训练集上进行。
