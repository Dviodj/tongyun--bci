## 🔧 技术方案

### 传统特征 + LDA (71.5%, 推荐)

**特征**:
1. mu频段(8-13Hz)功率
2. beta频段(13-30Hz)功率  
3. C3/C4不对称性 (运动想象核心特征)
4. 时域统计量

**分类器**: LDA (线性判别分析)

### EEGNet深度学习 (61.5%)

**架构**: EEGNet (Lawhern et al., 2018)
- 时域卷积 + 深度可分离卷积
- BatchNorm + Dropout正则化

**结论**: 小样本场景下不如传统方法

---

## 📖 关键发现

### 1. 数据泄露问题
早期99.1%结果源于：
- CSP在全部数据上拟合
- StandardScaler在全部数据上拟合

**已修复**: 所有特征在训练集上拟合

### 2. 深度学习 vs 传统方法
| 因素 | 传统方法 | 深度学习 |
|------|---------|---------|
| 小样本(120-160/人) | ✅ 稳定 | ❌ 易过拟合 |
| 特征工程 | 需要领域知识 | 自动学习 |
| 可解释性 | ✅ 高 | ❌ 低 |
| BCICIV 2b表现 | **71.5%** | 61.5% |

### 3. Cross-subject > Within-subject
每位被试样本太少，跨被试学习更有效。
